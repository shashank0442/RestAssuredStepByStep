# What is REST Assured | Use

1. REST Assured is a java library for testing RESTful web services.
2. It is used to invoke REST web services and check response.
3. Can be used to test XML as well as JSON based web services.
4. It supports POST, GET, PUT, DELETE, OPTIONS, PATCH and HEAD requests and can be used to validate and verify the
   response of these requests.
5. Can be integrated with testing frameworks like JUnit, TestNG etc.
6. REST Assured is implemented in Groovy.
